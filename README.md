# Rock Paper Scissors Game

An interactive browser-based Rock Paper Scissors game built with vanilla JavaScript.

## Features

- Click Rock, Paper, or Scissor to play against the computer
- Computer opponent picks randomly using Math.random()
- Game logic evaluates all win, lose, and draw conditions
- Result message displayed after every round
- Live scoreboard tracks user score and computer score across rounds
- Scores update dynamically without page reload

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
